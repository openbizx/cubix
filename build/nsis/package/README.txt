---------------------------------
Steps to start after installation
---------------------------------
1. Setup xampp 
Click Start Menu -> All Programs -> Openbizx-Cubi -> Setup Xampp
A command window is opened to execute xampp servers setup

2. Start xampp control panel
Click Start Menu -> All Programs -> Openbizx-Cubi -> Start Xampp Control Panel
Xampp control panel will be launched for user to manage apache and mysql services

3. Launch Openbizx Cubi
Click Start Menu -> All Programs -> Openbizx-Cubi -> Launch Openbizx Cubi
A browser will start to load local Openbizx Cubi url. E.g. http://localhost/cubi

---------------------------
Go to Openbizx Cubi website
---------------------------
Click Start Menu -> All Programs -> Openbizx-Cubi -> Website
A browser will start to load official openbiz website http://www.openbiz.me
